# Sugar Diary（砂糖日记）

Sugar Diary 是一款**本地优先的原生 Android 日记应用**。它适合想长期保存私人写作、照片、情绪记录，并希望在自己可控的 OpenAI 兼容接口上使用 AI 辅助整理、润色和心理洞察的用户。

> 当前版本：2.0.0

## 你可以用它做什么

- **写日记**：以 Markdown 为默认格式记录日常，也支持 HTML 内容查看与转换。
- **插入本地图片**：在编辑器光标位置嵌入设备图片，图片与正文一起保存在本地文章内容中。
- **查看和浏览记录**：通过时间轴、浏览页、搜索、标签、时间范围和心情标签找到过去的文章。
- **AI 格式转换**：把文章转换为目标格式，尽量保持原文语义不变。
- **AI 文风润色**：用内置或自定义文风提示词调整表达风格。
- **心理与情绪分析**：基于正文和图片线索生成单篇分析、建议、风险提示与历史记录。
- **多 Agent 心理洞察**：通过多个心理分析 Agent 分阶段处理文章，并保留过程摘要。
- **心理 Agent 对话**：围绕某篇日记继续追问和反思。
- **周期报告**：生成周报、月报，回顾情绪趋势和常见主题。
- **用户画像**：本地保存长期触发点、认知倾向、关系模式、资源优势等信息，使后续分析更贴近个人上下文。
- **数据迁移**：支持完整迁移包导入导出，也支持原始文章目录导入导出。

## 隐私与数据

Sugar Diary 默认不依赖云同步：

- 日记正文、图片、分析记录默认存储在设备本地。
- API Key 通过 Android 加密存储能力保存。
- Android 自动备份已关闭，降低本地隐私数据被系统备份带走的风险。
- 只有当你主动点击格式转换、润色、心理分析、周期报告或对话时，相关内容才会发送到你配置的 OpenAI 兼容接口。
- 图片不会把整段 base64 原文直接交给主模型；应用会先做图片摘要或引用替换，以降低超时和上下文膨胀风险。

如果你配置的是第三方模型服务，请同时阅读该服务的隐私政策。

## 页面说明

### 时间轴

- 查看全部日记。
- 创建新日记。
- 从最近更新时间进入编辑器。

### 浏览

- 按标题、日期、标签、时间桶、心情筛选文章。
- 预览正文摘要。
- 删除不需要的文章。

### 心理洞察

- 查看所有单篇心理分析。
- 进入分析详情、心理 Agent 对话、周期报告和用户画像。
- 重新触发单篇分析。

### 设置

- 配置主模型接口地址、模型名、API Key、超时时间。
- 配置独立图片理解模型。
- 修改心理分析提示词模板。
- 选择文章保存位置：应用私有目录或系统文件夹。
- 添加自定义文风。
- 导入导出完整迁移包或原始文章目录。

## AI 配置要求

应用使用 OpenAI 兼容 Chat Completions 接口。你需要准备：

- 接口地址：支持站点根地址、`/v1` 地址或完整 `/chat/completions` 地址。
- 主模型名称。
- API Key。
- 可选：独立图片理解模型地址、模型名和 API Key。

建议把请求超时设置在 15–300 秒之间；含图片或多 Agent 分析时可适当放宽。

## 安装与更新

1. 从 GitHub Releases 下载 APK。
2. 在 Android 设备上允许从当前来源安装应用。
3. 安装 APK。

如果你已经安装旧版，请确保新版 APK 使用同一签名，否则 Android 会拒绝覆盖安装。正式发布包应使用仓库维护者的 release keystore 签名。

## 开发者构建

普通用户不需要执行这些命令。开发者可在仓库根目录运行：

```bash
./gradlew testDebugUnitTest
./gradlew assembleDebug
./gradlew assembleRelease
```

Release 构建需要本地 `keystore.properties` 和签名材料；这些文件不会提交到仓库。
