# Sugar Diary

## 事先声明：完全AI编程，有任何问题都去骂codex

Sugar Diary（砂糖日记）是一款原生 Android 本地日记应用，面向希望长期保留私人写作记录、同时又需要 AI 辅助整理与分析的用户。应用支持本地写作、插入设备图片、OpenAI 兼容接口接入、格式转换、文风润色，以及围绕单篇日记与周期趋势展开的情绪分析能力。

## Core Features

- 本地优先：文章默认保存在设备本地，不依赖云端同步。
- 双格式工作流：正文默认创建为 Markdown，可在编辑页随时转换为 Markdown 或 HTML。
- 图片内嵌：可按光标位置插入本地图片，并在编辑区、查看页与摘要链路中正确处理。
- AI 格式转换：只负责把文章转换为目标格式，不改写原文语义。
- AI 文风润色：只调整表达风格，不擅自更改 Markdown / HTML / 普通文本结构。
- 情绪分析：结合正文与图片线索，生成情绪标签、强度、总结与支持性建议。
- 情绪中心：集中查看单篇分析、重新分析，并生成周期报告。
- 情绪详情页：针对单篇文章展示最新分析和历史分析记录。
- 周期报告：按周、月汇总情绪变化趋势。
- 文风系统：内置常用文风提示词，并支持自定义文风。
- 数据迁移：支持整库导入导出，也支持原始文章文件夹迁移。

## Navigation Overview

- `时间轴`：创建文章、查看全部记录时间线。
- `浏览`：按标题、日期、标签、时间桶、心情筛选文章，卡片显示正文摘要。
- `情绪`：查看周期报告、单篇分析列表，并进入独立情绪分析详情页。
- `设置`：配置 OpenAI 兼容接口、图片理解模型、提示词模板与存储位置。

## AI Features

- 支持自定义 OpenAI 兼容接口地址、模型名、超时秒数。
- 主模型与图片理解模型可分开配置。
- 默认情绪提示词已经升级为更强调共情、视觉线索与深层动机分析的多模态模板。
- 为了降低超时风险，内嵌图片在发送到模型前会先做摘要或占位符替换，不直接把整段 base64 原文发送给主模型。

## Privacy

- 日记正文与图片默认仅存储在本地。
- API Key 使用加密存储。
- 应用备份已关闭，避免系统备份直接带走本地隐私数据。
- 若接入第三方 LLM，只有你主动触发的内容才会发送到已配置接口。

## Project Structure

- `app/src/main/java/com/localdiary/app/data`: 数据存储、导入导出、LLM 接口与图片处理。
- `app/src/main/java/com/localdiary/app/model`: 领域模型与编辑器解析逻辑。
- `app/src/main/java/com/localdiary/app/ui`: Compose 界面、页面与 ViewModel。
- `app/src/test`: 解析、格式、筛选与报告生成相关单元测试。
- `app/src/main/icon`: 文本化图标源，构建时自动还原为启动图标。

## Build

```bash
./gradlew testDebugUnitTest
./gradlew assembleDebug
./gradlew assembleRelease
```

发布签名依赖本地 `keystore.properties`，该文件与签名材料默认不进入版本控制。
